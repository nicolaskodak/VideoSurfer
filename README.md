# VideoSurfer - 進階影片字幕播放器

一個功能豐富的網頁影片播放器，專門用於影片與字幕的同步播放、搜尋及導航。

## 功能特色

### 🎬 影片播放
- 支援多種影片格式（MP4、WebM、AVI 等）
- 響應式設計，適配桌面與行動裝置
- 高品質影片播放體驗

### 📝 字幕支援
- 支援 VTT 與 SRT 字幕格式
- 自動字幕格式轉換（SRT → VTT）
- 即時字幕同步顯示
- 支援中文字幕

### 🔍 智慧搜尋
- 即時字幕內容搜尋
- 關鍵字高亮顯示
- 一鍵跳轉至搜尋結果時間點
- 搜尋結果預覽

### 🎨 現代化介面
- 暗色主題設計
- 使用 Tailwind CSS 響應式佈局
- 直觀的使用者體驗
- 自訂滾動條樣式

## 使用方法

### 快速開始

1. **開啟應用程式**
   - 在瀏覽器中開啟 `index.html`
   - 無需安裝額外軟體或插件

2. **上傳影片**
   - 點擊「上傳影片」按鈕
   - 選擇您的影片檔案

3. **上傳字幕**
   - 點擊「上傳字幕」按鈕
   - 選擇 `.vtt` 或 `.srt` 字幕檔案

4. **搜尋字幕**
   - 在搜尋框中輸入關鍵字
   - 即時顯示搜尋結果
   - 點擊結果即可跳轉至對應時間

### 支援的檔案格式

#### 影片格式
- MP4
- WebM
- OGV
- AVI
- MOV
- 以及其他瀏覽器支援的格式

#### 字幕格式
- **VTT** (WebVTT) - 網頁原生支援
- **SRT** (SubRip) - 自動轉換為 VTT

## 功能詳解

### 字幕搜尋機制
1. **即時搜尋**: 使用者輸入時立即更新結果
2. **大小寫不敏感**: 自動忽略大小寫差異
3. **高亮顯示**: 搜尋關鍵字以黃色背景標示
4. **時間戳顯示**: 每個結果顯示對應的時間點

### 字幕格式轉換
- **SRT 轉 VTT**: 自動處理時間戳格式差異
- **編碼支援**: 支援 UTF-8 編碼
- **錯誤處理**: 優雅處理格式錯誤

### 記憶體管理
- **Object URL 清理**: 自動釋放不用的 URL 參照
- **軌道清理**: 移除舊字幕軌道避免衝突
- **效能最佳化**: 最小化記憶體佔用

## 最佳實踐

### 字幕檔案準備
1. **編碼格式**: 確保使用 UTF-8 編碼
2. **時間同步**: 字幕時間與影片精確對應
3. **檔案大小**: 建議字幕檔案小於 1MB

### 效能最佳化
1. **影片壓縮**: 使用適當的影片壓縮格式
2. **檔案大小**: 建議影片檔案小於 500MB
3. **網路環境**: 建議在本地環境使用以獲得最佳體驗

## 故障排除

### 常見問題

**Q: 字幕沒有顯示？**
- 確認字幕檔案格式正確（VTT 或 SRT）
- 檢查字幕檔案編碼為 UTF-8
- 重新上傳字幕檔案

**Q: 搜尋沒有結果？**
- 確認字幕已成功載入
- 檢查搜尋關鍵字拼寫
- 嘗試使用更短的關鍵字

**Q: 影片無法播放？**
- 確認瀏覽器支援影片格式
- 檢查影片檔案是否損壞
- 嘗試使用不同的影片檔案
