# VideoSurfer - AI 影片字幕分析播放器

一個融合 AI 技術的進階影片播放器，專門用於影片與字幕的同步播放、搜尋及智慧分析。使用 Google Gemini AI 提供強大的影片內容主題分析功能。

## 功能特色

### 🎬 影片播放
- 支援多種影片格式（MP4、WebM、AVI 等）
- 響應式設計，適配桌面與行動裝置
- 高品質影片播放體驗
- 精確的時間軸控制

### 📝 字幕支援
- 支援 VTT 與 SRT 字幕格式
- 自動字幕格式轉換（SRT → VTT）
- 即時字幕同步顯示
- 支援多語言字幕

### 🔍 智慧搜尋
- 即時字幕內容搜尋
- 關鍵字高亮顯示
- 一鍵跳轉至搜尋結果時間點
- 搜尋結果預覽

### 🤖 AI 主題分析
- **Google Gemini AI 整合** - 智慧分析影片內容
- **多語言支援** - 支援繁體中文、英文、日文、韓文、法文、西班牙文
- **主題分段** - 自動將影片分為 3 分鐘片段進行分析
- **主題摘要** - 為每個片段產生主題標題和描述
- **快速導航** - 點擊主題直接跳轉至對應時間點

### 🎨 現代化介面
- 暗色主題設計
- 使用 Tailwind CSS 響應式佈局
- 雙標籤頁設計（字幕搜尋 / 主題分析）
- 直觀的使用者體驗
- 自訂滾動條樣式

## 使用方法

### 快速開始

1. **開啟應用程式**
   - 在瀏覽器中開啟 `index.html`
   - 無需安裝額外軟體或插件

2. **上傳影片**
   - 點擊「上傳影片」按鈕
   - 選擇您的影片檔案

3. **上傳字幕**
   - 點擊「上傳字幕」按鈕
   - 選擇 `.vtt` 或 `.srt` 字幕檔案

4. **使用字幕搜尋**
   - 切換至「字幕搜尋」標籤頁
   - 在搜尋框中輸入關鍵字
   - 即時顯示搜尋結果
   - 點擊結果即可跳轉至對應時間

5. **AI 主題分析**
   - 切換至「主題分析」標籤頁
   - 輸入您的 Gemini API Key
   - 選擇分析語言
   - 點擊「分析主題」開始 AI 分析
   - 瀏覽主題摘要並快速導航

### Gemini API Key 設定

1. **取得 API Key**
   - 前往 [Google AI Studio](https://aistudio.google.com/)
   - 建立新專案或使用現有專案
   - 產生 Gemini API Key

2. **使用 API Key**
   - 在應用程式中貼上您的 API Key
   - API Key 僅在本次會話中使用，不會被儲存

## 功能詳解

### 雙標籤頁設計
1. **字幕搜尋標籤頁**
   - 即時關鍵字搜尋
   - 高亮顯示搜尋結果
   - 快速時間跳轉
   
2. **主題分析標籤頁**
   - AI 驅動的主題分析
   - 智慧片段分割
   - 主題摘要與導航

### 字幕搜尋機制
1. **即時搜尋**: 使用者輸入時立即更新結果
2. **大小寫不敏感**: 自動忽略大小寫差異
3. **高亮顯示**: 搜尋關鍵字以黃色背景標示
4. **時間戳顯示**: 每個結果顯示對應的時間點

### AI 主題分析流程
1. **片段分割**: 自動將字幕按 3 分鐘間隔分割
2. **平行處理**: 同時分析多個片段以提高效率
3. **智慧摘要**: 為每個片段產生主題和描述
4. **錯誤處理**: 個別片段失敗不影響整體分析
5. **多語言輸出**: 根據選擇的語言產生相應的分析結果

### 字幕格式轉換
- **SRT 轉 VTT**: 自動處理時間戳格式差異
- **編碼支援**: 支援 UTF-8 編碼
- **錯誤處理**: 優雅處理格式錯誤

### 記憶體管理
- **Object URL 清理**: 自動釋放不用的 URL 參照
- **軌道清理**: 移除舊字幕軌道避免衝突
- **效能最佳化**: 最小化記憶體佔用

## 最佳實踐

### 字幕檔案準備
1. **編碼格式**: 確保使用 UTF-8 編碼
2. **時間同步**: 字幕時間與影片精確對應
3. **檔案大小**: 建議字幕檔案小於 1MB
4. **內容品質**: 確保字幕內容清晰完整以獲得更好的 AI 分析效果

### Gemini API 使用
1. **API Key 安全**: 不要在公開場合分享您的 API Key
2. **使用限制**: 注意 Gemini API 的使用限制與計費
3. **語言選擇**: 選擇與字幕內容相符的分析語言
4. **網路環境**: 確保穩定的網路連接以獲得最佳 AI 分析體驗

### 效能最佳化
1. **影片壓縮**: 使用適當的影片壓縮格式
2. **檔案大小**: 建議影片檔案小於 500MB
3. **分析時機**: 在影片載入完成後再進行 AI 分析
4. **瀏覽器效能**: 建議在桌面瀏覽器上使用以獲得最佳體驗

## 故障排除

### 常見問題

**Q: 字幕沒有顯示？**
- 確認字幕檔案格式正確（VTT 或 SRT）
- 檢查字幕檔案編碼為 UTF-8
- 重新上傳字幕檔案

**Q: 搜尋沒有結果？**
- 確認字幕已成功載入
- 檢查搜尋關鍵字拼寫
- 嘗試使用更短的關鍵字
- 確認已切換至「字幕搜尋」標籤頁

**Q: 影片無法播放？**
- 確認瀏覽器支援影片格式
- 檢查影片檔案是否損壞
- 嘗試使用不同的影片檔案

**Q: AI 分析失敗？**
- 檢查 Gemini API Key 是否正確
- 確認網路連接穩定
- 檢查是否已上傳字幕檔案
- 確認 API Key 是否有足夠的使用額度

**Q: AI 分析結果不準確？**
- 確保字幕內容清晰完整
- 嘗試選擇與字幕內容相符的語言
- 檢查字幕時間戳是否正確

**Q: 分析速度過慢？**
- 檢查網路連接速度
- 較長的影片需要更多時間進行分析
- 可以考慮分批上傳較短的影片片段
